Probléma

Gyakorlati alkalmazhatóság

A Firefox OS környezetben elsajátítótt mobil alkalmazás fejlesztői ismereteket nem csak egy, hanem jelenleg az össze elterjedt okos telefon platformon felhasználhatjuk. Ha esetleg néhány év múlva úgy döntünk, hogy felhaygunk a Firefox OS akalmazások fejlesztésével, bátran igent mondhatunk bármilyen web front-end-es munka lehetőségre is, hiszen a Firefox OS környezetben megszerzett tudásunkat és tapasztalatunkat könnyedén tudjuk majd kamatoztatni.
