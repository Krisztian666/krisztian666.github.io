Új ismeretek

Alkalmazásunk Web Storage API segítségével lesz képes kulcs / érték párok tárolására. Ez a megoldás sokkal intuitívabb mintha cookie-kat használnánk. A Web Storage API két tárolás mechanizmust támogat. Minden alkalmazás számára futás során elérhető egy elszeparált sessionStorage tároló, ez egy olyan ideiglenes tároló melybe az alkalmazásaink szabadon írhatnak adatokat. A tárolt adatok mmegmaradnak az alkalmazás vagy oldalaik újratölrése után is A localStorage is ugyanazt csinálja, de az itt tárolt adatok akkor is megmaradnak, ha az alkalmazás lezárul és újra indul. A tárolókat a Window.sessionStorage és Window.localStorage hívásokon keresztül érhetjük el. A tárolóban adatokat elhelyezni setItem lekérdezni a getItem metódusokkal tudunk.
