Új ismeretek

A felhasználók számára teljes szerkesztő nézetet az Edit mode komponens segítségével biztosíthatunk. A komponens használatához a edit_mode.css állományot kell az alkalmazás oldalaink számára elérhetővé tenni. Az Edit mode alkalmzásával egy olyan szerkesztő nézetet biztosíthatunk alkalmazásunk használóinak mely tartalmaz egy teljes értékű fejlécet, és egy gombsávot is. Az megjelenő elemek egy FORM elem foglaja magába. A fejléc a már megismert SECTION segítségével a gombsáv pedig a MENUben elhelyezett BUTTON elemekből áll.
