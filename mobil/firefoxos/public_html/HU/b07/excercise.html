Kidolgozott mintapélda

Cseréljük le a ToDo alkalmazásunk nyitó oldalát egy Layout alapú megoldásra.

Gyakorló feladatok

  • Cseréld le az órarend kezelő alkalmazásod nyitó oldalát egy Layout alapú megoldásra.
  • Cseréld le a kölcsönadott tárgyak nyilvántartásást megvalósító alkalmazásod nyitó oldalát egy Layout alapú megoldásra.
  • Cseréld le a családi bevásárló listát kezelő alkalmazásod nyitó oldalát egy Layout alapú megoldásra.
