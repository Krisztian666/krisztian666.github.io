Új ismeretek

A IDBObjectStore interfész valósítja meg az IndexedDB API-ban az Objektumok tárolását egy adatbázisban. Az objektumok tárolását rendezett kulcsokkal valósítja meg. Ez teszi lehetővé a gyors feltöltést, keresést, és lekérdezést.

IDBObjectStore.put Returns an IDBRequest object, and, in a separate thread, creates a structured clone of the value, and stores the cloned value in the object store. This is for updating existing records in an object store when the transaction's mode is readwrite.

IDBObjectStore.getAll returns an IDBRequest object retrieves all objects in the object store matching the specified parameter or all objects in the store if no parameters are given.

The IDBRequest interface of the IndexedDB API provides access to results of asynchronous requests to databases and database objects using event handler attributes. Each reading and writing operation on a database is done using a request.

Event handlers

IDBRequest.onerror The event handler for the error event.

IDBRequest.onsuccess The event handler for the success event.
