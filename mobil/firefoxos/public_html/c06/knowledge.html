Új ismeretek

A IDBObjectStore interfész valósítja meg az IndexedDB API-ban az Objektumok tárolását egy adatbázisban. Az objektumok tárolását rendezett kulcsokkal valósítja meg. Ez teszi lehetővé a gyors feltöltést, keresést, és lekérdezést.

IDBObjectStore.put Visszaadd egy IDBRequest példányt, ezen objektum példány eseménykezelőin keresztül érjük el az adatmódosítás eredményét.

Az IndexedDB API az IDBRequest interfész segítségével asszinkron módon biztosít hozzáférést a eredményeihez.
